一、缘起

  话说第5天明确了需求后，我就开始着手页码器组件的开发了。对于组件的样式，一开始我是准备不使用 UI 库的。一方面是因为之前使用 Semantic-UI 留下了阴影（理性来讲，其实更多的是其 API 造成的，样式倒还好），另一方面则是想到自己不就是在开发一个未来 UI 库的基本组件嘛，再去用别人的库总有一种没有自主知识产权的味道。后来一琢磨，想想我只要用它的 segment 样式就好了，单独引用进来，然后如果有需要再用自己的样式表对其进行覆盖就好了。
