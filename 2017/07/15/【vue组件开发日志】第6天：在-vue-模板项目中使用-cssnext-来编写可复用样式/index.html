发表于 2017-07-15

Try Editor.md

一、背景

  在 vue 项目模板中，比如我们写了一个样式，并希望这个样式能在任意组件中复用，我们可以把这个样式写在一个单独的 *.css 文件中，然后在需要的时候 import *.css 即可， webpack 会将 *.css 中的样式生成一个 style 标签插入到组件的 html 中。如果是 *.sass 或 \.less 等，vue 项目模板中的 webpack 配置还会支持将它们转化成 css 后再 import。

二、问题

  但问题是，我尝试了 *.postcss ，在里面写 cssnext 语法的样式，然后在 App.vue 或 main.js 中 import ，却总是得到没有解析的的代码。你也可以试试。
  那么最后我是如何将 cssnext 语法写的样式独立出来后，import 进去并转化成正确的 css 样式的呢？
